# Cowork Bootcamp — Session 4: Project Kit & Presentations

One call becomes a proposal, deck, LinkedIn post, and follow-up email. Your agent handles the packaging.

## Install

Add the marketplace, then install the plugin:

```
/plugin marketplace add heymitch/cowork-bootcamp-session-4
/plugin install cowork-bootcamp-session-4@speakeasy-session-4
```

One install — all three skills appear automatically.

## Skills

| Skill | Trigger | What It Does |
|-------|---------|-------------|
| Project Kit | "Build a project kit from this call" | Turn a meeting into a proposal + deck + post + email |
| Gamma Connector | "Connect Gamma" | Set up Gamma for AI-powered presentations and decks |
| Social Content | "Write a LinkedIn post about X" | Generate social media posts for LinkedIn and Twitter |

## Session Order

1. **Gamma Connector** — connect Gamma for presentation generation (5 minutes)
2. **Project Kit** — turn a call or meeting into a full deliverable package
3. **Social Content** — create posts to share your work
